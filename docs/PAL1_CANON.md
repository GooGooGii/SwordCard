# 仙劍奇俠傳 1 設定考據集（PAL1 Canon Reference）

## 一、四位主角

| 角色 | 出身 | 武器 | 代表招式 / 仙術 | 合擊技 |
|---|---|---|---|---|
| **李逍遙** | 餘杭鎮客棧，外婆（李大娘）撫養的孤兒 | 劍（木劍→鐵劍→龍泉劍…最強刀為鬼牙刀） | 御劍術、萬劍訣、天師符法、天劍、劍神（雷/土系） | 合體氣功 |
| **趙靈兒** | 仙靈島，**女媧後裔（人蛇化身）**，巫后林青兒之女 | 杖（巫女專用；天蛇杖限女媧後人）、越女劍 | 靈族神術、治療系、五靈法術 | 爆炸 |
| **林月如** | **靈劍山莊**，南武林盟主**林天南**之女，嬌縱大小姐 | 劍 / 刀（鳳鳴刀；玄冥寶刀） | 劍術、刀法、弦月斬 | 弦月斬 |
| **阿奴** | 南詔國苗族，**聖姑（祖母）**之孫女 | 杖（巫月神刀為其終極武器） | 蠱術、毒術、苗疆秘法 | 天女散花 |

**師承**：李逍遙的啟蒙恩師是 **酒劍仙**（蜀山派掌門之師弟），傳授御劍術。

**關鍵設定**：
- 趙靈兒是**蛇 / 水 / 女媧**主題；林月如是**劍**；阿奴是**蠱毒**；李逍遙是**御劍 + 雜學（外婆的藥方）**。
- 靈珠系統：玉佛珠 / 風 / 雷 / 水 / 火 / 土 / 聖靈珠，裝備可用對應系法術。

## 二、重要 NPC / 配角

| 角色 | 設定 | 結局 / 關係 |
|---|---|---|
| **酒劍仙** | 蜀山派，李逍遙恩師，邋遢醉道士、朱漆酒葫蘆 | 傳逍遙御劍術；斬殺蜘蛛精毒娘子 |
| **唐鈺（小寶）** | **大理城漢族**，與**阿奴青梅竹馬**，佩**青釭劍**（非青鋼劍）| 與阿奴情深；電視劇版兩人化為比翼鳥 |
| **林天南** | 南武林盟主，林月如之父 | 辦比武招親（逍遙誤上擂台勝月如）|
| **巫后（林青兒）** | 趙靈兒之母，仙靈島巫族 | 犧牲自己封印水魔獸 |
| **拜月教主** | 黑苗族邪教魔魁，傀儡術 / 驅魔獸；曾私養水魔獸嫁禍巫后 | 最終與水魔獸結合，趙靈兒與之同歸於盡 |
| **劉晉元** | 新科狀元，**林月如表哥**，一直喜歡月如 | 中纏魂絲毒，被蝶妖彩依所救 |
| **彩依** | 修行千年的**蝴蝶精**，報恩於救命書生劉晉元 | 散盡千年道行換晉元十年陽壽，重新化回蝴蝶（配樂《蝶戀》）|
| **毒娘子** | 蜘蛛精，纏魂絲劇毒之主 | 由李逍遙、林月如所斬 |

## 三、主要地點（依劇情順序）

| 順序 | 地點 | 重點事件 |
|---|---|---|
| 1 | **餘杭鎮** | 逍遙的家鄉客棧；過十里坡到山神廟學劍法 |
| 2 | **十里坡** | 練功起點（蜜蜂等雜兵；「十里坡劍神」迷因出處）|
| 3 | **仙靈島 / 水月宮** | 趙靈兒出浴初遇；取紫金丹 |
| 4 | **蘇州城** | 遇林月如、劉晉元；比武招親擂台 |
| 5 | **將軍塚** | 試煉 boss 戰地點 |
| 6 | **試煉窟** | 藏全遊戲最強五靈法術（底層）|
| 7 | **鎖妖塔** | 揭露靈兒人蛇身；boss **鎮獄明王**；龍柱盤洞妖龍 |
| 8 | **南詔（南紹）** | 巫后被誣陷；巫王已被拜月教主所殺、由樹妖假扮 |
| 9 | **拜月教壇** | 最終決戰；水魔獸 + 拜月教主 |

> SwordCard 五幕對照：餘杭山間 / 蘇州地底 / 苗疆蠱土 / 鎖妖塔 / 拜月決戰。

## 六、關鍵劇情 / 名場面（可作事件素材）

- **十里坡劍神**：玩家在十里坡苦練打蜜蜂的迷因 → 已用於 `shilipo_sword_god` 事件
- **比武招親**：逍遙誤上林家擂台勝月如 → 可作事件
- **蝶戀（彩依）**：蝶妖救書生、散盡道行 → 已用於 `caiyi_butterfly` 事件 +「百花仙釀」藥品
- **酒劍仙傳藝**：→ 已用於 `drunk_swordsman` 事件
- **唐鈺求劍 / 阿奴情緣**：→ 已用於 `tangyu_sparring` 事件
- **巫后犧牲封印水魔獸**：尚未取用，可作高張力事件
- **趙靈兒同歸於盡**：結局向，慎用
